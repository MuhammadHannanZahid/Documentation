  ## Page Identity:
  The Orion Intelligence authentication portal is the starting/entry point of the web application. It has a dual-purpose design(left and right pane) that act as both, User login authentication and a summary of what this application is capable of, in cyber-intelligence/cyber-security.  

  1. **Application Name:** Orion Intelligence.
  2. **Design:** It consists of deep navy and sky-blue colours showing *Dark Mode* vibe conveying a message of security and technical environment.
  3. **Layout:** It consists of a split-pane design that separates the user interaction options from the informational content representing what this application is about.

  ## Technical stack:

  ## Authentication portion(The left pane):
    This portion consists of the "Input Controls/Options" which are required from the user for the system access.

    1. **Branding:** The "Orion Intelligence" logo is centered with a call-to-action "Login to get started!".

    2. **Input Fields:** This is the place where users enter login credentials. It consists of 2 parts,
      - **Email:** place where users(who already have account) enter their email address.
      - **Password:** place where users(who already have account) enter their password followed by a Reset Password option indicated by "Reset Password?" question(Place where users can reset their password if they forget it).

    3. **User Action:** The portal consists of a high contrast "Sign In" button and a question(Don't have an account?) following by a "Sign up" action button which gives an option to the new user to create their own account using this link ([Sign up](https://try.orionintelligence.org/signup)).

    4. **Top Navigations:** There are three links that are Top-aligned,
      - **Privacy Policy:** Gives information about the terms of use, privacy rights and data transparency. 
      - **Our Services:** Gives information about what we provide users with like Network Intelligence, Database security, Web Protection, etc.
      - **About us:** Provides information about who we are, what we do, what do we consist of, etc. 
      
      For further information follow these links,
      - ([Privacy Policy](https://drive.google.com/file/d/16NQysXxTdEg5Jx6KNok-rqHvdeM-YOJO/view))
      - ([Our Services](https://orionintelligence.org/service/))
      - ([About us](https://orionintelligence.org/))

  ## System Preview(Part of right Pane):
    Gives an image of what is inside the web application and in how much depth the work is being done in this web application.

    1. **Analytics Dashboard:** The visual representation of the "generic index", showing statistical bar charts for "Top Teams" Leak vs Defacement.
    2. **Live Feeds:** A display showing current "Top Data Leaks" with clear display of source, the company whose data was leaked and on which date it leaked.
